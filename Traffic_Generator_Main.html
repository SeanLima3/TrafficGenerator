Pro Tip 
  [
    {
      "role": "agent",
      "verbiage": "Thank you for calling Xfinity technical support. My name is Alex. May I have your name, please?"
    },
    {
      "role": "patron",
      "verbiage": "Hi Alex, this is Sarah. I'm having issues with my Xfinity Internet. It's been really slow lately."
    },
    {
      "role": "agent",
      "verbiage": "I apologize for the inconvenience, Sarah. Let me check your connection. Are all the lights on your modem stable?"
    },
    {
      "role": "patron",
      "verbiage": "Yes, all the lights are on, but the internet speed is frustratingly slow, especially during peak hours."
    },
    {
      "role": "agent",
      "verbiage": "I understand, Sarah. Let me run a quick diagnostic on your connection. It seems there might be some congestion in your area. I'll optimize your settings for better performance. Please bear with me for a moment."
    },
    {
      "role": "patron",
      "verbiage": "Sure, take your time. I appreciate your help."
    },
    {
      "role": "agent",
      "verbiage": "Thank you for your patience, Sarah. I've made the adjustments. Please restart your modem, and let me know if you notice any improvement in your internet speed."
    },
    {
      "role": "patron",
      "verbiage": "Alright, I'll do that now. Thank you, Alex, for your assistance."
    },
    {
      "role": "agent",
      "verbiage": "You're welcome, Sarah. If the issue persists, don't hesitate to call us back. Have a great day!"
    }
  ]
      
  This is an example of what you can ask GPT to create a suitable json file:
  
  Generate a transcript of a technical support call for Xfinity Internet. The call should include an 'agent' role and a 'patron' role and be 60 seconds in length. The output must conform to the following json schema:
  
  {
    "$schema": "http://json-schema.org/draft-04/schema#",
    "type": "array",
    "items": [
      {
        "type": "object",
        "properties": {
          "role": {
            "type": "string"
          },
          "verbiage": {
            "type": "string"
          }
        },
        "required": [
          "role",
          "verbiage"
        ]
      }
    ]
  }
      
  Did you know you can add pauses (and change other things like rate of speech, etc.) using SSML markup in your json file?     
   
  Just put the speak tags around the text, and then where you want the pause, add <break time=5s/> - replacing 5 with how many seconds you want to wait.
   
  Example: 
   
  {
      "role": "agent",
      "verbiage": "<speak>Thank you for calling.  Please give me a moment to review your account.<break time=5s/> Ok, I found your account.  What can I help with today?</speak>"
    }
